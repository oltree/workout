# Project Description

This App is a backend application for the Workout project.

## Technologies

- Express
- JavaScript
- PostgreSQL + Prisma
- @trivago/prettier-plugin-sort-imports + Prettier
- Cors

## Installation

1. Clone the repository.
2. Navigate to the project directory.
3. Run `npm i` to install the dependencies.
